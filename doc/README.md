# doc/ — project documentation

| Folder | What's inside |
|--------|---------------|
| [`reference/`](reference/) | **Current, authoritative guides.** Architecture handbook, complete system guide, execution playbook, operations manual, parity & design reviews, build status, verification report. Start here. |
| [`pdf/`](pdf/) | Printable/shareable PDFs — the system guide (generated by `scripts/generate_system_guide.py`), operations manuals, and rebuild SOPs. |
| [`assets/`](assets/) | Diagrams and the scrape-cost calculator spreadsheet (generated by `scripts/scrape_cost_calculator.py`). |
| [`legacy/`](legacy/) | **Historical** docs from the earlier 4-SKU / 33-cell era, before the 585-cell scale-up. Kept for provenance; treat as history, not current truth. |

The repo's front-page overview is the root [`README.md`](../README.md). Per-run reports live under `v4_outputs/<timestamp>/` and `DISCOUNT_PLAN/`, not here.
